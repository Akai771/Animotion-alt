# Animotion
**Anime streaming, news, and AI recommendations — all in one place.**

Animotion is a free anime streaming platform built for enthusiasts. Watch thousands of titles, stay updated with the latest episodes and news, and get personalized recommendations from Hiro — an AI chatbot that actually knows anime.

## Features

- **Free Streaming** — Access a vast library of anime titles, no subscription required
- **Latest Episodes & Schedules** — Stay updated with newest episodes and airing times
- **Anime News** — Real-time news pulled from Anime News Network
- **Hiro — AI Chatbot** — Personalized recommendations and in-depth anime insights powered by OpenAI
- **Random PFP Generator** — Generate random anime profile pictures

## Tech Stack

| | |
|---|---|
| Framework | React 19 + TypeScript |
| Build | Vite |
| Styling | Tailwind CSS + Shadcn + MagicUI |
| Backend | Supabase (user data & preferences) |
| AI | OpenAI API (Hiro chatbot) |
| Anime Data | [Aniwatch API](https://github.com/ghoshRitesh12/aniwatch-api) + [Anify API](https://docs.anify.tv/) |
| News | Anime News Network API |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Akai771/AniMotion-alt.git
cd Animotion

# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build
```
